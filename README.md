Tic Tac Toe game with a modern UI, smooth animations, optional sound, and a scoreboard. Code is split into HTML, CSS, and JavaScript files.

## How to Run

Open the project in any browser:

1. **Recommended:** serve the folder locally so `style.css` and `script.js` load reliably:
   - `python3 -m http.server 8000`
   - Open: `http://localhost:8000`
2. Or open `index.html` directly (double-click). If styles/scripts do not load, use step 1.

## Features

- 3x3 Tic Tac Toe grid
- Two modes:
  - Two Players (X vs O)
  - Vs Computer (you are X, AI is O)
- Win / draw detection with animated winning line highlight
- Restart/Reset round button
- Scoreboard (X wins, O wins, draws)
- Smooth cell mark animation + hover effects
- Dark/Light mode toggle
- Optional sound effects (click + win/draw)
- Responsive layout (works on mobile and desktop)

## Files

- `index.html` - semantic markup and structure
- `style.css` - layout, themes, animations
- `script.js` - game logic, AI, sound, scoreboard
- `assets/readme-banner.png` - README banner image (optional)
