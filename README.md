# Tic Tac Toe (HTML/CSS/JavaScript)

Single-file Tic Tac Toe game with a modern UI, smooth animations, optional sound, and a scoreboard.

## How to Run

Because this is a plain `index.html`, you can run it in any browser:

1. Open `index.html` by double-clicking it, **or**
2. Serve the folder locally (recommended):
   - `python3 -m http.server 8000`
   - Open: `http://localhost:8000`

## Features

- 3x3 Tic Tac Toe grid
- Two modes:
  - Two Players (X vs O)
  - Vs Computer (you are X, AI is O)
- Win / draw detection with animated winning line highlight
- Restart/Reset round button
- Scoreboard (X wins, O wins, draws)
- Smooth cell mark animation + hover effects
- Dark/Light mode toggle
- Optional sound effects (click + win/draw)
- Responsive layout (works on mobile and desktop)

## Files

- `index.html` - all markup, styling, and JavaScript (single file)
